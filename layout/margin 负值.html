应用2：两端对齐

场景：从后台拿到数据后，自动生成横向列表(数量未知)，列表之间保持一定间距(margin-right相同)，但是最后一项不能看到右边距，即实现两端对齐。

原理：当子元素不设置宽度时(width: auto)，边距越大，宽度越小；边距越小，宽度越大。

应用3：等高对齐

  • 原理：边距能改变元素占位大小（width + padding + border + margin） 背景色可以在padding里显示，可以为每一个列表项都设置一个很大的padding值用以显示背景色值然后用margin负值抵消，这样就不会改变占位大小（还是元素真实内容的大小）。
  • 场景：右边内容较少，高度撑不起来，但希望背景色与左边等高，因此加大padding值

应用4：垂直水平居中

如果要居中的元素的宽/高是不变的或者说是确定的，比如width/height=100px，那么设置absolute的top/left=50%，然后margin-left/margin-top=-50px即可

如果要居中的元素的宽/高是不确定的，这时margin负值就不能使用具体的px了，可以使用百分比。但由于margin的百分比都是相对于父元素的宽度，利用绝对定位的包裹性，在目标元素外面再包一层absolute定位的父元素，可以实现两层元素等宽等高

特点3：元素加宽

如果元素的宽度没有设置（width: auto, 即由浏览器自动计算出实际宽度），左右负边距会把元素向两个方向拉以增加宽度。与正边距的效果一样：当宽度自动时，边距越大，宽度越小；边距越小，宽度越大。应用：两端对齐

公式：父元素width = 子元素width + padding + border + margin

在这里，父元素width = 300，子元素margin = -10, border = 2, padding = 0 => 子元素width = 308px, 从而实现了元素加宽
